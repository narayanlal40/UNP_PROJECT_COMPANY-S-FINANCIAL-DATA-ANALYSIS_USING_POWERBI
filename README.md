# UNP_PROJECT_COMPANY-S-FINANCIAL-DATA-ANALYSIS_USING_POWERBI

This Power BI project analyzes financial companies’ data to evaluate key performance indicators such as revenue, profit, expenses, and growth trends. It uses interactive dashboards to compare different companies, track financial performance over time, and identify high-performing sectors. The report helps in understanding financial stability and supports better decision-making through clear and meaningful visual insights.
